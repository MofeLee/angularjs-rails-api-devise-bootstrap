== AngularJS Single Page Devise Authentication (with Bootstrap)

Ruby on Rails starter project with Devise Authentication on Single Page AngularJS app. Client and server are cleanly separated and communicate using JSON Restful contract.

The following libraries are used:

- Rails-API
- Gulp (for client site workflow)
- AngularJS
- UI-Router
- ng-token-auth
- Devise token auth 
- Twitter Bootstrap
